# Ideas

This is a scratchpad. Dump half-formed thoughts, feature ideas, questions for future-you, things you don't want to forget. Nothing here is a commitment — it's a place to think out loud.

When an idea is ready to become real work, select the text in the editor and use the right-click menu to convert it into a plan or a backlog item. The original note stays here for history.

## Example: dark mode toggle

What if the editor supported a true black OLED mode, not just dark-gray? I keep hearing this from people on new phones / laptops. Would it be one color variable swap, or do we have hardcoded grays scattered around? Worth a quick scan of `global.css` before committing.
